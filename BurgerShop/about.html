About This Project

Burger Shop is a full-stack web project built using Node.js, Express.js, db.json(File-based data handling using JSON), and XOR Authentication. It simulates an online burger ordering platform where users can browse products, add them to the cart, and place orders, while admins can manage the products and view orders.

🎯 Key Features

  • 🔐 Admin/User authentication (Signup, Login, Logout)
  • 🔐 Admin/User is remembered by storing email on local storage
  • 🧑‍💼 Admin can add product-related data (name, price, addons, etc.)
  • 📊 Admin can view and manage orders from all users
  • 📈 Admin dashboard includes graphical representation and analytics of sales
  • 🛍️ Users can view, add, and remove products from the cart
  • 📦 Users can place orders; admins can view them
  • 🧾 Bills are generated for placed orders
  • 🧠 Passwords are encrypted using simple XOR
  • 🗂️ Simple db.json file is used for storing users, products, and orders

🛠️ Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: File-based data handling using JSON
  • Authentication: XOR operator and equality check
  • Frontend: HTML, CSS, JavaScript
  • Storage: Public folder for images, JS, and CSS files
